ol(Ordered List) : 순서가 있는 목록

type 속성 : 표시되는 숫자 모양을 변경
type="a, A"; : 영어 소, 대문자로 나타남
type="i, I" : 로마자로 표시
start 속성 : 시작 숫자 지정
reversed 속성 : 숫자를 역순부터 시작

  2. 책상
  3. 의자
  4. 조명
